# 🌌 Иерархия Вселенной · Cosmic Hierarchy

> Интерактивное дерево иерархии существ, рас и сознаний вселенной.  
> Архив структур бытия — от Абсолюта до Аватара.

## 🚀 Запуск на GitHub Pages

1. Загрузите все файлы в репозиторий
2. Перейдите в **Settings → Pages**
3. Source: **Deploy from a branch** → **main** → **/ (root)**
4. Сайт будет доступен по адресу: `https://ВАШ_ЛОГИН.github.io/ВАШ_РЕПОЗИТОРИЙ`

## 🎮 Как пользоваться

- **Нажмите "ВОЙТИ"** на экране входа — начнёт играть музыка
- **Кликните на любой блок** дерева — откроется детальное описание
- **Кнопка ♪ Музыка** — включить/выключить музыкальную тему
- **Escape** или клик вне рамки — закрыть описание
